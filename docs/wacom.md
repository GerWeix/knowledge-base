# Wacom Tipps

* Pinselgröße/härte verändern: CTRL+ALT\-Rechtsklick. Größe: re-li, Härte: oben-unten
* Opacitiy immer auf 100%, mit _Fluss_ arbeiten
* Mit ALT Farbe aufnehmen
* Brush-Popup deaktivieren: WIN **> Stift- und Fingereingabe (pen and touch) > Gedrückt halten (Press and hold) > Einstellungen >** `Gedrückthalten für Rechtsklick` deaktivieren
* How to remove the splash circle in windows 7
    1. WIN+R and type `gpedit.msc` into the command line
    1. Navigate to… **User Configuration > Administrative Templates > Windows Components > Tablet PC > Cursor**.
    1. Set `Turn off pen feedback` to enabled.
    1. Now navigate to… **Computer Configuration > Administrative Templates > Windows Components > Tablet PC > Cursor**.
    1. And again set `Turn off pen feedback` to enabled.
