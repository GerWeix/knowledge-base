# Wacom Tipps

* Pinselgröße/härte verändern: <kbd>CTRL</kbd>+<kbd>ALT</kbd>-Rechtsklick. Größe: re-li, Härte: oben-unten
* Opacitiy immer auf 100%, mit _Fluss_ arbeiten
* Mit ALT Farbe aufnehmen
* Brush-Popup deaktivieren: <kbd>WIN</kbd> **> Stift- und Fingereingabe (pen and touch) > Gedrückt halten (Press and hold) > Einstellungen >** `Gedrückthalten für Rechtsklick` deaktivieren
* How to remove the splash circle in windows 7
  * <kbd>WIN</kbd>+<kbd>R</kbd> and type `gpedit.msc` into the command line
  * Navigate to… **User Configuration > Administrative Templates > Windows Components > Tablet PC > Cursor**.
  * Set `Turn off pen feedback` to enabled.
  * Now navigate to… **Computer Configuration > Administrative Templates > Windows Components > Tablet PC > Cursor**.
  * And again set `Turn off pen feedback` to enabled.
